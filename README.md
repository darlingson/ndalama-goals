# NdalamaGoals

NdalamaGoals is a native Android application designed to help users track their personal savings goals. It provides a clean, modern interface for managing and visualizing progress towards achieving financial objectives.

## Features

- **Goal Management:** Create, view, and manage multiple savings goals.
- **Contribution Tracking:** Log contributions towards each goal to monitor progress.
- **Visual Progress:** See a detailed view of each goal, including the current balance, target amount, and progress indicators.
- **Modern UI:** Built with Jetpack Compose and Material 3 for a beautiful and responsive user experience.

## Screens

- **Goals List:** A dashboard showing a summary of all goals, including a priority goal and a list of active goals.
- **Goal Details:** A detailed view of a single goal, showing its progress, contribution history, and options to edit or pause the goal.
- **Create Goal:** A form to create a new savings goal with a name, target amount, and target date.
- **Add Contribution:** A screen to add a financial contribution to a specific goal, with options for amount, category, and notes.

## Technologies Used

- **Kotlin:** The primary programming language for the application.
- **Jetpack Compose:** For building the entire user interface declaratively.
- **Material 3:** The design system used for UI components and styling.
- **AndroidX Libraries:** A suite of libraries for core application functionality, including Navigation for in-app navigation.
- **Gradle:** For dependency management and building the project.

## Getting Started

To build and run this project, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```

2.  **Open in Android Studio:**
    Open the project in the latest version of Android Studio.

3.  **Sync Gradle:**
    Let Android Studio sync the Gradle files and download all the required dependencies.

4.  **Run the application:**
    Select an emulator or a physical device and click the "Run" button in Android Studio. The application will be built and installed on the selected device.

    Alternatively, you can build the application from the command line using Gradle:
    ```bash
    ./gradlew assembleDebug
    ```
